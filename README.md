# java-stack

## 简介

**建项原因**

系统化的学习知识，有利于将知识碎片拼凑起来，融会贯通。这对于掌握任何一门技术，都是非常必要的。所以，作为一名 Java 工程师，应该系统化管理自己的 Java 技术栈知识。

**范围**

学海无涯，学的越多，越感到个人精力有限，难以对所有技术都面面俱到。所以，根据自己的工作领域，有侧重点的去掌握知识才是明智的。

由于本人人从事 Java Web 领域的开发，所以本项目主要是整理 Java Web 及相关领域的知识点。
